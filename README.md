# 🔬 PubMed to BibTeX Manager

![Version](https://img.shields.io/github/v/release/andreavalsesia/pubmed-to-bibtex)
![Version](https://img.shields.io/github/license/andreavalsesia/pubmed-to-bibtex)
![Version](https://img.shields.io/badge/Biber%20Ready%20-%20?link=https%3A%2F%2Fbiblatex-biber.sourceforge.net)

**PubMed to BibTeX Manager** is a lightweight but powerful browser extension built for researchers and students. It extracts accurate metadata directly from PubMed and generates clean citations compatible with BibTeX and biber. The clean interface keeps track of all your saved papers directly in the browser and it's perfect for managing bibliography in any small or big project. 

---

## ✨ Main Features

- **Intelligent Extraction**: Gets citation information from page meta-tags.
- **Advanced APA style names formatting**: It automatically converts complex names (ex. "De Costa MW" ➔ "De Costa, M. W.").
- **Local bibliographic library**: Save your papers permanently in your browser. No accounts needed!
- **PMID Link**: Reopen the original PubMed page from your library.
- **Backup & Sync**: Export your library as a JSON file and reimport it on any device (or read it as-is even without the extension).
- **Privacy First**: Your data remain local insider your `local-storage`.

## 🚀 How To Install (Quickstart)

This extension is under development and open-source. You can install it manually in your browser as follows:

### For Chrome, Edge and Chromium browser
1. Download the latest release file `pubmed-to-bibtex-chrome.zip` from [**Releases**](https://github.com/andreavalsesia/pubmed-to-bibtex/latest).
2. Extract the ZIP file on your PC.
3. Open Chrome and go to `chrome://extensions/` or find the option "Manage extensions" in the user interface.
4. Enable **Developer Mode** (on top-right corner).
5. Click on **Load unpackaged** and select the previously extracted folder.

### For Firefox
1. Download `pubmed-to-bibtex-firefox.zip` from [Releases](https://github.com/andreavalsesia/pubmed-to-bibtex/latest).
2. Open Firefox and go to `about:debugging#/runtime/this-firefox`.
3. Click on **Load temporary component**.
4. Select the ZIP file (without extracting).
